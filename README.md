# Python Data Analysis: Sales Data

## Project Overview
This project performs data analysis using Python. Pandas and Matplotlib were used to clean, manipulate, and visualize sales data.

## Files
- `sales_analysis.ipynb` : Jupyter Notebook with analysis and visualizations.
- `sales_data.csv` : Dataset used for analysis.

## Key Insights
- Identified sales trends over time.
- Analyzed top-performing regions and products.
- Visualized data for clear decision-making support.

## Tools Used
- Python
- Pandas
- Matplotlib
